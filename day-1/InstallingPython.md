#  Installing Python

Before starting the lessons, you need to install Python on your computer.

---

#  Windows Installation

## Step 1 — Download Python

Go to the official Python website:

[Python Official Website](https://www.python.org/downloads/?utm_source=chatgpt.com)

Click **Download Python**.

---

## Step 2 — Run the Installer

Open the downloaded file.

⚠️ IMPORTANT:
Before clicking install, check the box:

```text
Add Python to PATH
```

Then click:

```text
Install Now
```

---

## Step 3 — Verify Installation

Open:

```text
Command Prompt (CMD)
```

Type:

```bash
python --version
```

If Python is installed correctly, you will see something like:

```bash
Python 3.13.0
```

---

#  Recommended Code Editor

You can write Python code using:

* Visual Studio Code
* PyCharm
* IDLE (comes with Python)

---

##  Installing Visual Studio Code

Download VS Code here:

[Visual Studio Code](https://code.visualstudio.com/?utm_source=chatgpt.com)

After installing:

1. Open VS Code
2. Go to Extensions
3. Search for:

```text
Python
```

4. Install the Python extension

---

#  Running Your First Python Program

Create a file called:

```text
hello.py
```

Add this code:

```python
print("Hello, World!")
```

Save the file.

Open the terminal and run:

```bash
python hello.py
```

---

#  Expected Output

```text
Hello, World!
```

---

#  Note

If `python` does not work, try:

```bash
py --version
```

or

```bash
python3 --version
```

depending on your system.
