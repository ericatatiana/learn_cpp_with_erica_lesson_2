---

##  What is Hello World?

In programming, the first program beginners usually create is called **Hello World**.

It is a simple program used to display a message on the screen.

---

##  Example

```python
print("Hello, World!")
```

---

##  Explanation

* `print()` is a Python function used to display information on the screen.
* `"Hello, World!"` is the message being displayed.

---

## ▶️ Output

```python
Hello, World!
```
