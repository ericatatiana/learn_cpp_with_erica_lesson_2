## What are Variables?

Variables are used to store data in a program.

You can think of a variable as a container that holds information.

---

## 💻 Example

```python
name = "Erica"
age = 100

print(name)
print(age)
```

---

##  Explanation

* `name` stores text (`"Erica"`)
* `age` stores a number (`100`)
* `print()` displays the values on the screen

---

##  Output

```python
Erica
100
```

---

#  Variable Types

Python supports different types of data.

## String

Used for text.

```python
city = "Maputo"
```

---

## Integer

Used for whole numbers.

```python
year = 2026
```

---

## Float

Used for decimal numbers.

```python
price = 19.99
```

---

## Boolean

Used for `True` or `False`.

```python
is_student = True
```

---

#  Multiple Variables Example

```python
name = "Tiana"
country = "Mozambique"
age = 100
height = 1.65

print(name)
print(country)
print(age)
print(height)
```
